使用了將近兩年的 Medium，最近決定從 Medium 遷移至 gatsby 中，為什麼呢？

# 遷移到哪？

將遷移到 gatsby，並使用 github 作為主要 hosts

# 為什麼遷移？

一種感覺問題（尤其付費牆大量出來以後），Medium 漸漸的不是這麼吸引我了。

# 有什麼壞處

- 少了流量
- 需要自行管理

# 有什麼好處

- 有更多彈性
